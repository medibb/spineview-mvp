# 제품 요구사항 정의서 (PRD)
## 요추-골반 굴곡-신전 분석 대시보드

---

## 1. 프로젝트 개요

### 1.1 목적
Movella DOT IMU 센서에서 수집된 CSV 데이터(척추 및 골반)를 분석하여,
굴곡-신전(Flexion-Extension) 각도 및 다양한 생체역학적 지표를 시각화하는
Django 기반 웹 대시보드를 구축합니다.

### 1.2 배경
- 요추-골반 협응(Lumbo-Pelvic Rhythm)은 허리 건강 및 움직임 품질을 평가하는 핵심 지표
- IMU 센서 기반의 동작 분석은 비용 효율적이고 접근성이 높음
- 직관적인 시각화를 통해 연구자 및 임상가가 데이터를 쉽게 해석할 수 있도록 지원

### 1.3 범위
**포함 (In Scope)**
- CSV 파일 업로드 및 파싱
- 쿼터니언 → 오일러 각도 변환
- 굴곡-신전 각도 계산 및 시각화
- 통계 분석 및 요약 지표 제공
- 반응형 웹 대시보드

**제외 (Out of Scope)**
- 실시간 스트리밍 데이터 처리
- 데이터베이스 영구 저장
- 임상 등급의 생체역학적 정확도 검증
- 사용자 인증/권한 관리

## 2. 입력 데이터 사양

### 2.1 파일 구성
| 파일명 | 설명 | 부착 위치 |
|--------|------|-----------|
| spine_dot.csv | 척추(요추) 센서 데이터 | L3-L4 척추 부위 |
| pelvis_dot.csv | 골반 센서 데이터 | 천골(Sacrum) 부위 |

### 2.2 CSV 구조 (Movella DOT 형식)

실제 데이터는 메타데이터 블록 없이 첫 행부터 헤더로 시작합니다.

#### 실제 데이터 예시
```csv
PacketCounter,SampleTimeFine,Quat_W,Quat_X,Quat_Y,Quat_Z,Acc_X,Acc_Y,Acc_Z,Gyr_X,Gyr_Y,Gyr_Z,Status,
0, 158174677, -0.1048, -0.6171, 0.2037, -0.7528, 0, 0, 0, 0, 0, 0, 0,
1, 158191344, -0.1051, -0.6170, 0.2047, -0.7526, 9.564, -2.028, 2.095, -5.288, -0.663, 2.416, 0,
```

#### 데이터 헤더 및 컬럼
| 컬럼명 | 데이터 타입 | 설명 | 필수 여부 |
|--------|-------------|------|-----------|
| PacketCounter | Integer | 패킷 순번 (0부터 시작) | ⬜ 선택 |
| SampleTimeFine | Integer | 고정밀 타임스탬프 (μs) | ✅ 필수 |
| Quat_W | Float | 쿼터니언 W 성분 (스칼라) | ✅ 필수 |
| Quat_X | Float | 쿼터니언 X 성분 | ✅ 필수 |
| Quat_Y | Float | 쿼터니언 Y 성분 | ✅ 필수 |
| Quat_Z | Float | 쿼터니언 Z 성분 | ✅ 필수 |
| Acc_X | Float | 가속도 X (m/s²) | ✅ 활용 |
| Acc_Y | Float | 가속도 Y (m/s²) | ✅ 활용 |
| Acc_Z | Float | 가속도 Z (m/s²) | ✅ 활용 |
| Gyr_X | Float | 각속도 X (deg/s) | ✅ 활용 |
| Gyr_Y | Float | 각속도 Y (deg/s) | ✅ 활용 |
| Gyr_Z | Float | 각속도 Z (deg/s) | ✅ 활용 |
| Status | Integer | 센서 상태 (0=정상) | ⬜ 선택 |

### 2.3 데이터 특성 (실측)
```
📊 spine_dot.csv 분석 결과
├── 총 샘플 수: 893개
├── 샘플링 주기: ~16,667 μs (약 60Hz)
├── 총 측정 시간: 약 14.9초
├── 쿼터니언 크기 (|q|): ~1.0 (정규화됨)
└── 데이터 품질: 양호 (결측치 없음)
```

### 2.4 시간 동기화
- 두 센서의 `SampleTimeFine` 값이 정확히 일치하지 않을 수 있음
- 시간 보간(interpolation)을 통해 동기화 필요
- 샘플링 레이트: 60Hz (16.667ms 간격)

### 2.5 파서 요구사항
- 첫 행을 헤더로 인식
- 후행 쉼표 처리 (trailing comma)
- 공백 트리밍 (데이터 앞뒤 공백 제거)
- 다양한 구분자 지원 (쉼표, 세미콜론, 탭)
- 인코딩: UTF-8
- 누락된 값 처리 (보간 또는 제외)

## 3. 계산 로직

### 3.1 쿼터니언 → 오일러 각도 변환

```
Roll (X)  = atan2(2(qw·qx + qy·qz), 1 - 2(qx² + qy²))
Pitch (Y) = asin(2(qw·qy - qz·qx))  ← 굴곡-신전
Yaw (Z)   = atan2(2(qw·qz + qx·qy), 1 - 2(qy² + qz²))
```

### 3.2 굴곡-신전(FE) 각도 계산

| 지표 | 공식 | 설명 |
|------|------|------|
| FE_spine(t) | Pitch 각도 (척추) | 척추 센서의 시상면 각도 |
| FE_pelvis(t) | Pitch 각도 (골반) | 골반 센서의 시상면 각도 |
| Relative_FE(t) | FE_spine(t) − FE_pelvis(t) | 요추-골반 상대 각도 |

### 3.3 파생 지표 계산

#### 3.3.1 시간 영역 분석
| 지표 | 공식 | 의미 |
|------|------|------|
| ROM (Range of Motion) | max(θ) - min(θ) | 총 운동 범위 |
| 평균 각도 | mean(θ) | 평균 자세 위치 |
| 표준편차 | std(θ) | 움직임 변동성 |
| 최대 굴곡 | max(θ) | 최대 굽힘 각도 |
| 최대 신전 | min(θ) | 최대 펴짐 각도 |

#### 3.3.2 속도 분석 (Gyr 데이터 활용)
| 지표 | 공식 | 의미 |
|------|------|------|
| 평균 각속도 | mean(\|ω\|) | 평균 움직임 속도 |
| 최대 각속도 | max(\|ω\|) | 최고 움직임 속도 |
| 각속도 RMS | √(mean(ω²)) | 움직임 강도 지표 |
| 시상면 각속도 | Gyr_Y | 굴곡-신전 속도 (주요 분석 대상) |

#### 3.3.3 가속도 분석 (Acc 데이터 활용)
| 지표 | 공식 | 의미 |
|------|------|------|
| 벡터 크기 | √(Acc_X² + Acc_Y² + Acc_Z²) | 총 가속도 크기 |
| 가속도 변동성 | std(Acc_magnitude) | 움직임 불안정성 |
| 저크(Jerk) | d(Acc)/dt | 움직임 부드러움 지표 |

#### 3.3.4 요추-골반 협응 분석
| 지표 | 공식 | 의미 |
|------|------|------|
| 기여도 비율 | FE_spine ROM / Relative_FE ROM | 척추의 상대적 기여도 |
| 위상 관계 | cross-correlation | 동시성/순차성 분석 |
| 협응 지수 | R² (회귀분석) | 움직임 동기화 정도 |

### 4.2 주요 시각화 컴포넌트

#### 컴포넌트 1: 굴곡-신전 시계열 그래프 (메인)
- **유형**: 인터랙티브 라인 차트
- **라이브러리**: Plotly.js (권장) 또는 Chart.js
- **기능**:
  - 3개 신호 동시 표시 (척추 FE, 골반 FE, 상대 FE)
  - 줌/팬 인터랙션 (마우스 드래그/휠)
  - 범례 클릭으로 개별 신호 표시/숨김
  - 호버 시 정확한 값 + 타임스탬프 표시
  - 구간 선택 기능 (범위 분석용)
  - 크로스헤어 커서

#### 컴포넌트 2: 각속도 시계열 (Gyr)
- **유형**: 멀티 라인 차트
- **표시 항목**: Gyr_X, Gyr_Y, Gyr_Z
- **특별 표시**: Gyr_Y (시상면 각속도) 강조
- **활용**: 움직임 속도 패턴 분석

#### 컴포넌트 3: 가속도 시계열 (Acc)
- **유형**: 멀티 라인 차트  
- **표시 항목**: Acc_X, Acc_Y, Acc_Z + 벡터 크기
- **활용**: 움직임 강도 및 안정성 분석

#### 컴포넌트 4: 통계 요약 카드
- **유형**: 카드형 KPI 표시
- **레이아웃**: 그리드 (3열: 척추/골반/상대)
- **표시 항목**:
  | 지표 | 척추 | 골반 | 상대 |
  |------|------|------|------|
  | ROM | ○ | ○ | ○ |
  | 평균 | ○ | ○ | ○ |
  | 표준편차 | ○ | ○ | ○ |
  | 최대값 | ○ | ○ | ○ |
  | 최소값 | ○ | ○ | ○ |
  | 피크 각속도 | ○ | ○ | - |

#### 컴포넌트 5: 분포 분석
- **유형**: 히스토그램 + 박스플롯 (수직 결합)
- **목적**: 각도 분포 특성 파악
- **기능**:
  - 빈(bin) 수 슬라이더 조절
  - 정규분포 곡선 오버레이 (선택)
  - 분포 선택 드롭다운 (척추/골반/상대)
  - Shapiro-Wilk 정규성 검정 결과 표시

#### 컴포넌트 6: 협응 분석 (산점도)
- **유형**: 산점도 + 회귀선
- **X축**: 골반 FE 각도
- **Y축**: 척추 FE 각도
- **표시**: 
  - 선형 회귀선 (최소제곱법)
  - R² 결정계수
  - 95% 신뢰구간 (음영)
  - 잔차 분포 (미니 히스토그램)

#### 컴포넌트 7: Angle-Angle Plot (위상 궤적)
- **유형**: 패스 플롯 (경로 시각화)
- **X축**: 골반 FE 각도
- **Y축**: 척추 FE 각도
- **목적**: 굴곡-신전 사이클의 위상 패턴 분석
- **표시**:
  - 시작점 ● / 끝점 ■ 마커
  - 방향 화살표 (진행 방향)
  - 색상 그라데이션 (시간 진행: 파랑→빨강)
  - 주요 이벤트 포인트 (최대 굴곡/신전)

#### 컴포넌트 8: 메타데이터 푸터
- **유형**: 정보 바
- **표시**:
  - 총 측정 시간
  - 총 샘플 수
  - 샘플링 레이트
  - 파일명
  - 분석 완료 시간
